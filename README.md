# 📝 AI Text Recognition System using Python, OpenCV and Tesseract OCR

## 📌 Project Overview

This project is an AI-powered Text Recognition System developed as part of my Artificial Intelligence Internship at DecodeLabs.

The system extracts text from images using Optical Character Recognition (OCR) technology. It reads text from an image, displays the extracted content, and saves it into a text file.
## 🚀 Features

✅ Image-to-Text Conversion

✅ Optical Character Recognition (OCR)

✅ Text Extraction from Images

✅ Automatic Text File Generation

✅ OpenCV Image Processing

✅ Tesseract OCR Integration
## 🛠️ Technologies Used

- Python
- OpenCV
- Pytesseract
- Tesseract OCR Engine

## ⚙️ How It Works

### Step 1
The user provides an image containing text.

### Step 2
The image is loaded using OpenCV.

### Step 3
The image is converted to grayscale for better OCR accuracy.

### Step 4
Tesseract OCR extracts the text from the image.

### Step 5
The extracted text is displayed on the screen.

### Step 6
The extracted text is saved into a text file.
